## 2026-05-07 — 행사 회차·일정 전면 정정

| 항목 | 잘못된 값 (정정 전) | 정식 값 (정정 후) |

| 회차 | 제41회 / 41st edition | 제42회 / 42nd edition |
| 연도 | 2026 | 2027 |
| 일정 (한국어) | 2026년 3월 19일(목) ~ 22일(일) | 2027년 3월 18일(목) ~ 21일(일) |

| 가이드 버전 | v2026.0 | v2027.0 |

| 행사 종료 시점 | 2026년 3월 22일 | 2027년 3월 21일 |
| 해시태그 | `#KIMES2026` | `#KIMES2027` |

### 적용 범위

명세서 안의 모든 다음 표기를 일괄 치환:

- "KIMES 2026" → "KIMES 2027"
- "KIMES2026" (해시태그) → "KIMES2027"
- "제41회" → "제42회"
- "41st edition" → "42nd edition"

- "2026년 3월 19일", "March 19, 2026" 등 모든 날짜 표기 → 2027 일정으로
- "2026년 41회", "2026년에는 41회" → "2027년 제42회"

명세서 §5.3.4의 "각 숫자 옆 작은 글씨로 '2026 기준 (잠정)' 표기" 규칙은 다음과 같이 갱신:

### 코드·인프라 영향

- **components/config.js** 신규 — `window.KIMES_EVENT` 객체로 회차·일정·버전·숫자 데이터의 단일 출처 확보. 다음 회차 갱신 시 이 파일 한 곳만 수정.
- **components/AppShell.jsx** — 푸터의 버전·갱신일을 `KIMES_EVENT`에서 읽도록 변경.
- **index.html** — 다른 컴포넌트보다 먼저 config.js 로드.

### 변경되지 않은 항목

- 명세서 작성일 (`2026-05-07`) — 명세 자체의 작성 시점이므로 그대로.
- 1980년 (KIMES 역사상 최초 개최 연도) — 사실 데이터, 변경 없음.
- 변경 이력 예시 중 "2026-04-15", "2026-03-20" 등의 과거 일자 — 가이드 갱신 타임라인으로 그대로 보존.
