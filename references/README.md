## KIMES 가이드 톤 (이게 우선)

KIMES BI 가이드는 다음 결로 작성합니다.

- 영문 H1 + 한글 서브 (§22.5)
- 카피는 짧게, 한 줄 라벨 위주 (토스·네이버 결)
- 카드 분량은 자주 쓰는 케이스만 (배민·당근 결)
- 위협조 단어 회피 (절대·반드시·금지·위반·불가 — §22.6)
- 산업 전시 진중함 유지 (B2B 톤)

레퍼런스가 추가되더라도 가이드 전체 톤을 통째로 따라가지 않습니다.
부분적 시스템·시각 처리만 차용합니다.
