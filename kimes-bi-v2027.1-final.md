## §3. 로고 (/logo)

**eyebrow** 02 · **H1** 로고

**lede** KIMES 로고 파일과 사용 규정. 변형·재타이핑·임의 색상 변경은 권장하지 않습니다.

### 3.1 로고

[KIMES 로고 큰 이미지]

KIMES 로고는 1980년 첫 행사 이래 사용해왔습니다. 빨간색·두꺼운 산세리프·`i` 안 검은 쐐기 — 이 세 요소가 KIMES 로고의 정체성입니다.

### 3.2 로고 버전

SVG는 디지털·웹, PNG는 미리보기·메신저, AI/EPS는 인쇄·간판용.

연도가 들어간 로고(예: KIMES 2027 일체형)는 사무국에 요청해주세요 (kimes@kimes.kr).

| 버전 | 용도 | 다운로드 |
|---|---|---|
| 기본 (레드) | 흰 배경·밝은 사진 위 — 가장 일반적인 사용 | SVG / PNG / AI / EPS |
| 화이트 | 빨간색·검정·어두운 배경 위 | SVG / PNG / AI / EPS |
| 블랙 | 단색 인쇄·팩스·복사 | SVG / PNG / AI / EPS |
| 그레이 | 본문 안 작은 표기·각주 | SVG / PNG / AI / EPS |

해상도: 벡터 / 컬러모드: SVG = sRGB · AI = CMYK + Pantone 185 C 별색

### 3.3 여백

로고 사방으로 K 높이만큼 여백 확보. 여백 안에는 다른 로고·문구·이미지를 두지 않습니다.

[다이어그램: 로고 + 사방 1X 점선 박스]

**1X = K의 높이** · 점선 안쪽 여백

### 3.4 최소 크기

다음 크기 이상으로 사용해주세요. 더 작아지면 `i` 안 쐐기가 뭉개집니다.

| 매체 | 최소 크기 | 사용 자리 |
|---|---|---|
| 디지털 | 80 px | 이메일 서명·웹 푸터·SNS 카드 등 화면 표기 |
| 인쇄 | 24 mm | 명함·리플렛·리포트 표지 등 종이 매체 |
| 사이니지·대형 인쇄 | 제한 없음 | 단, 명도·해상도 사전 확인 권장 |

### 3.6 피해야 할 사용 (7가지)

KIMES 로고 파일은 변형 없이 그대로 사용해주세요. 글자 표기는 별개 영역 → /typography.

1. **색 변경** — 회사 컬러로 통일하지 않습니다
2. **크기 변형 (가로·세로 늘이기)** — 비율을 변형하지 않습니다
3. **로고 위 텍스트·그래픽 겹치기** — 로고 위에 다른 요소를 얹지 않습니다
4. **색 반전** — KIMES Red·흰·검 외 색 반전을 피해주세요
5. **화질 낮은 이미지 사용** — SVG 또는 고해상도 PNG를 사용해주세요
6. **비공식 로고 사용** — /logo에서 받은 공식 SVG만 사용해주세요
7. **회사 로고와 어색한 배치** — 시각적 무게가 비슷하도록 배치해주세요

**Q1. KIMES 로고 색을 우리 회사 컬러에 맞춰 바꿔도 되나요?**
권장하지 않습니다. KIMES 로고는 KIMES Red, 흰색, 검정 중 하나로
사용해주세요. 회사 브랜드 컬러로 KIMES를 표기하고 싶다면 로고 파일
대신 글자 표기 (→ /typography §8.4)를 활용하세요.

**Q2. KIMES 로고 안의 검은 쐐기를 빼도 되나요?**
권장하지 않습니다. 로고의 핵심 요소이므로 그대로 사용해주세요.

**Q3. SVG 파일 외에 PNG로만 받을 수 있나요?**
모든 버전이 SVG·PNG·AI·EPS로 제공됩니다. /downloads에서 받으세요.

**Q4. 로고를 더 큰 해상도로 받을 수 있나요?**
SVG와 AI는 벡터이므로 무한 확대 가능합니다. PNG는 최대 4000px 폭으로
제공됩니다.

### 컬러 (Q5–Q6)

**Q5. KIMES Red의 정확한 Pantone 번호는?**
Pantone 185 C (코팅) / 185 U (비코팅).

**Q6. 인쇄소가 CMYK로 0/100/100/0을 찍었는데 색이 살짝 어둡습니다.**
종이 종류(코팅·비코팅)와 인쇄기에 따라 차이가 있습니다. Pantone 185 C
별색 매칭을 권장합니다.

### 표기 (Q7–Q9)

**Q7. "KIMES"를 "킴스"로 한글 표기해도 되나요?**
권장하지 않습니다. 영문 KIMES 그대로 사용해주세요.

**Q8. "제42회"를 "42nd"로 영문 표기할 때 형식은?**
"KIMES 2027 — 42nd edition" 형식.

**Q9. 보도자료에 KIMES 풀네임을 매번 써야 하나요?**
처음 등장할 때만 풀네임. 두 번째부터는 "KIMES"만 사용.

### 결합 사용 (Q10–Q13)

**Q10. 부스 헤더에 우리 회사 로고와 KIMES 로고 비율은?**
회사 로고와 비슷한 시각적 무게로 배치합니다. 픽셀 비율보다 눈으로
확인하세요 (참고: 약 1 : 0.7).

**Q11. 이메일 서명에 KIMES 로고를 넣고 싶은데 양식이 있나요?**
/downloads의 Marketing 카테고리에서 이메일 서명 배너 4종을 받으실 수
있습니다 (Outlook · Gmail · Apple Mail · 모바일).

**Q12. 명함에 KIMES 표기를 넣어도 되나요?**
행사 기간 한정으로 가능. 행사 종료 후 재인쇄 시 제거 권장.

**Q13. SNS 게시물에 KIMES 로고 들어간 걸 올릴 때 별도 승인 필요한가요?**
일반적인 게시물은 별도 승인 불필요. 캡션에 @kimes_official 태그를
권장합니다 (의무 아님 — /notation §9.2.7 SNS Tag 참고).

### 권한·승인 (Q14–Q19)

**Q14. KIMES 로고가 들어간 굿즈를 만들어도 되나요?**
사전 승인 필요. kimes@kimes.kr.

**Q15. 회사 광고에 KIMES 로고를 넣고 싶습니다.**
사전 승인 필요. 매체·기간·시안 첨부.

**Q16. "KIMES 공식 파트너"라고 표기해도 되나요?**
별도 라이선스 계약이 필요한 표현입니다. 사용 전 사무국과 협의해주세요.

**Q17. 사장님이 KIMES 사진을 회사 인터뷰에 쓰고 싶어 합니다.**
사진 종류·노출 매체에 따라 다름. kimes@kimes.kr 문의.

**Q18. 영상 인트로에 KIMES 로고 1초 노출은 승인 필요한가요?**
노출 시간과 무관하게 사전 승인 필요.

**Q19. 행사 끝나고도 회사 홈페이지에 KIMES 참가 표기를 유지해도 되나요?**
차회 참가가 확정되지 않았다면 "Past Exhibitor" 표기로 변경 또는 제거.

### 특별관 (Q20–Q21)

**Q20. 우리는 MedicomteK 특별관에 참가합니다. KIMES 로고를 안 써도 되나요?**
외부 자료(보도자료·홈페이지)에는 KIMES 로고를 함께 표기합니다. 특별관 단독
표기는 특별관 안에서만.

**Q21. 특별관 로고는 어디서 받나요?**
/special-zones에서 각 특별관별로 다운로드.

### 다운로드 (Q22–Q23)

**Q22. 로고 파일 압축 해제가 안 됩니다.**
한국어 파일명 인코딩 문제일 수 있습니다. 다른 압축 해제 프로그램(반디집·
7-Zip)을 써보세요.

**Q23. AI 파일을 못 여는데 다른 형식 있나요?**
SVG·EPS로 받아 사용하거나 무료 도구(Inkscape)로 열 수 있습니다.

### 행사 일반 (Q24–Q29)

**Q24. 부스 위치는 어떻게 정해지나요?**
참가 신청 순서·카테고리·이전 회차 참가 이력에 따라 사무국이 배정.

**Q25. 사전등록 마감일은?**
2027년 3월 17일 23:59 (행사 시작 전날).

**Q26. 부스 설치 가능 시간은?**
행사 전일 09:00~21:00. 자세한 운영 매뉴얼은 참가업체 매뉴얼 참조.

**Q27. 참가업체 출입증 발급은?**
사전등록 시 자동 발급. 행사 첫날 East Gate 참가업체 데스크 수령.

**Q28. 외국인 참가업체 비자 지원이 있나요?**
초청장 발급 가능. kimes@kimes.kr 문의.

**Q29. 가이드에 없는 사용 케이스가 있습니다.**
kimes@kimes.kr 문의. 영업일 3~5일 안에 답변.
