## 3. Logo (`/#/logo`)

### 3-1. Wordmark 본문 톤

기존
> KIMES 로고는 1980년 첫 행사 이래 단단하고 직선적인 산세리프로 한국 의료기기 산업의 무게감을 표현해왔습니다. 강렬한 빨강과 두꺼운 글자, 그리고 ¡ 안에 자리 잡은 검은 쐐기 — 이 세 요소가 KIMES만의 시각적 정체성입니다.

변경
> KIMES 로고는 1980년 첫 행사부터 사용된 산세리프 워드마크. 빨강·블랙 두 색과 ¡ 안의 쐐기가 핵심 요소.

### 3-2. Versions 본문 톤

기존
> 용도에 맞는 버전을 선택해 다운로드하세요. SVG는 디지털·웹 표준, PNG는 빠른 미리보기·예시지 첨부, AI/EPS는 인쇄·간판 작업용입니다.
>
> 행사 직전 발표용 연도 일체형 로고(예: KIMES 2027 일체형)는 사무국이 별도로 제공합니다. 필요한 경우 kimes@kimes.kr로 요청해주세요.

변경
> 용도별 버전. SVG는 디지털·웹, PNG는 미리보기, AI는 인쇄·간판용.
>
> 연도 일체형 로고는 별도 제공. 필요 시 kimes@kimes.kr 요청.

### 3-3. Versions EPS 칩 삭제

**대상**: 4종 버전(기본/화이트/블랙/그레이) 카드 내 파일 형식 칩

**기존**: `SVG  PNG  AI  EPS`
**변경**: `SVG  PNG  AI`

각 카드의 `EPS` 칩만 삭제. SVG/PNG/AI 다운로드는 활성화 유지.

### 3-4. Minimum Size 본문 톤

기존
> 다음 크기 이상에서만 KIMES 로고를 사용합니다. 더 작아지면 ¡ 안의 검은 쐐기가 뭉개져 식별이 어려워집니다.

변경
> 최소 크기 미만 사용 금지. ¡ 안의 쐐기 식별성 저하.

### 3-5. Background Use "상세 사양" 블록 삭제

**유지**: 본문 + "빠른 참고" 표 + "Brand Color Backgrounds" 5색 그리드

**삭제**:
- "상세 사양" 헤더
- "Primary Wordmark (Red)" 그라데이션 바 + 설명 ("명도 10% 이하 흰색에 빨간 배경에서 사용..." 등)
- "Black / White Wordmark" 그라데이션 바 + 설명 ("0~30% 백색: Black 로고 / 40~100% 백색: White 로고")

### 3-6. Don'ts 본문 톤

기존
> KIMES 로고 SVG는 변형하지 않고 그대로 사용합니다. 텍스트 표기는 마케팅·콘텐츠 영역의 별개 작업입니다 — 권장 사양·표기 사례는 → /typography.

변경
> 로고 SVG는 원본 그대로 사용. 텍스트 표기는 → /typography 참고.

### 3-7. Request Approval 본문 톤

기존
> 가이드에 명시된 일반적인 사용은 별도 신청이 필요 없습니다. 다만 다음 경우는 사무국 사전 승인을 받아주세요.

변경
> 일반 사용은 신청 불필요. 다음의 경우 사전 승인 필요.

## 5. Typography (`/#/typography`)

### 5-1. 리드문 톤

기존
> KIMES를 자기 자료에 표기할 때 두 가지 파일(로고 SVG vs 텍스트 표기)을 어떻게 다르게 쓰는지, 권장 서체와 사양은 무엇인지 안내합니다.

변경
> KIMES 표기용 파일 두 종(로고 SVG / 텍스트) 구분 및 권장 서체·사양 안내.

### 5-2. Fonts 본문 톤

기존
> KIMES는 Montserrat(영문) + Pretendard(한글) 페어를 권장합니다. 둘 다 무료·오픈소스. 굵기는 800 ExtraBold 또는 700 Bold를 권장 합니다 — 부스·인쇄물에서 멀리서도 또렷.

변경
> 권장 서체: Montserrat(영문) + Pretendard(한글). 굵기 800 ExtraBold 또는 700 Bold. 둘 다 무료·오픈소스.

### 5-3. Recommended Notation 본문 톤

기존
> 회사 폰트가 따로 있다면 그대로 사용해주세요. 회사 디자인 시스템이 따로 없는 경우 아래 사양을 참고하시면 결이 자연스럽습니다.

변경
> 회사 폰트가 있으면 우선 사용. 없을 경우 아래 사양 참고.

### 5-4. Wordmark Use 본문 톤

기존
> 공식 보증·증명이 필요한 자리(보도자료 헤더 마크, 부스 인증 마크, 공문 헤더 등)에는 KIMES 로고 SVG를 사용해주세요. 다른 폰트로 "KIMES"를 타이핑해서 로고 자리에 박는 것은 피해주세요. 그 외 마케팅·콘텐츠 텍스트 표기는 자유롭게 사용하셔도 됩니다.

변경
> 공식 보증·증명 자리(보도자료 헤더, 부스 인증, 공문 헤더 등)는 로고 SVG 사용. 다른 폰트로 "KIMES" 타이핑해 로고 자리에 사용 금지. 일반 텍스트 표기는 자유.

변경
> KIMES 언급 시 표기 규칙. 명칭·SNS 태그·승인 필요 표현 등. 보일러플레이트는 → /overview 참고.

### 6-2. Name Notation 표 — 영문 정식 변경

**위치**: "처음 언급" 행

**기존 OK**: `KIMES 2027 (Korea International Medical & Hospital Equipment Show)`
**변경 OK**: `KIMES 2027 (The 42nd Korea International Medical & Hospital Equipment Show)`

(나머지 행은 변경 없음)

### 6-3. Edition 섹션 표 형식으로 재구성

**기존**: 불릿 리스트
- 한국어: 제42회 KIMES
- 영문: KIMES — 42nd edition
- 회차 숫자는 항상 아라비아 숫자 (한자·한글 사용하지 않음)

**변경**: 표 형식

| 항목 | 표기 |
|---|---|
| 권장 | KIMES 2027 / KIMES |
| 국문 정식 | 제42회 국제의료기기·병원설비전시회 |
| 영문 정식 | The 42nd Korea International Medical & Hospital Equipment Show |

### 6-4. Date & Time 섹션 전체 삭제

**삭제 대상**:
- "Date & Time" 헤더
- 부제 "날짜·시간"
- 표 전체 (단일 날짜 / 요일 포함 / 날짜 범위 / 월 범위 / 시간 / 구분기호 / 미권장)

**대체 안내**: Name Notation 표 하단에 한 줄 미주 추가
> 날짜·시간 표기는 → /overview의 Date & Venue 사용.

### 6-5. Place & Booth 표 재구성

**기존 4행**: 장소 / 홀 / 부스 / 한글 표기

**변경 2행**

| 항목 | ✓ OK | ✗ 미권장 |
|---|---|---|
| 장소 | 코엑스 / COEX | 씨오이엑스 / Coex / coex |
| 홀 | Hall A · Hall B-E | A Hall / 에이홀 |

**삭제**: "부스" 행, "한글 표기" 행

### 6-6. Numbers 섹션 전체 삭제

**삭제 대상**:
- "Numbers" 헤더
- 부제 "숫자"
- 4개 카드 (천 단위 콤마 / 퍼센트 / 가격 / 약식 큰 수)

### 6-7. Separators 섹션 전체 삭제

**삭제 대상**:
- "Separators" 헤더
- 부제 "구분 기호"
- 본문 안내
- 5행 표 (em-dash / en-dash / middle dot / slash / colon)

### 6-8. SNS Tag 본문 톤

변경

**변경**: Name Notation / Edition / Place & Booth / SNS Tag / Restricted Expressions

## 7. Special Zones (`/#/special-zones`)

### 7-1. 리드문 톤

기존
> KIMES 2027은 3개의 특별관을 운영합니다. 각 특별관은 KIMES 안에서 별도 테마와 시각 정체성을 갖습니다. 특별관 참가업체는 KIMES 표기에 더해 해당 특별관 표기도 함께 사용할 수 있습니다.

변경
> KIMES 2027은 3개 특별관 운영. 특별관 참가업체는 KIMES 표기와 해당 특별관 표기를 병기.

### 7-2. 특별관 3종 — 로고 자산 행 정리

**대상**: MedicomteK / BEAUTY&DERMA SEOUL / INSPIRE Digital Health 카드 모두

**기존**: `SVG  PNG  AI  EPS    Coming Soon`
**변경**: `SVG  PNG  AI`

- `EPS` 칩 삭제
- `Coming Soon` 텍스트 삭제
- SVG / PNG / AI 다운로드 활성화

### 7-3. 특별관 한 줄 본문 톤 (3종 공통)

**MedicomteK**

기존
> MedicomteK은 KIMES 안의 특별관으로 의료 커뮤니케이션·커넥티드 케어 기술을 모은 전시입니다. 의료 IT·원격진료·의료 통신 솔루션 회사들이 참가합니다.

변경
> 의료 커뮤니케이션·커넥티드 케어 기술 전시. 의료 IT·원격진료·통신 솔루션 회사 참가.

**BEAUTY&DERMA SEOUL**

기존
> BEAUTY&DERMA SEOUL은 KIMES 안의 특별관으로 미용의학·피부과 전문 전시입니다. 서울·부산 두 에디션으로 운영되며 미용기기·피부과 솔루션 회사들이 참가합니다.

변경
> 미용의학·피부과 전문 전시. 서울·부산 두 에디션 운영. 미용기기·피부과 솔루션 회사 참가.

**INSPIRE Digital Health**

기존
> INSPIRE Digital Health는 KIMES 안의 특별관으로 디지털 헬스·웨어러블·헬스케어 AI 솔루션 전시입니다. 디지털 헬스 스타트업·플랫폼 회사들이 참가합니다.

변경
> 디지털 헬스·웨어러블·헬스케어 AI 솔루션 전시. 디지털 헬스 스타트업·플랫폼 회사 참가.

### 7-4. 하단 안내 톤

기존
> 특별관 단독 표기는 특별관 안에서만 사용합니다. 외부 자료(보도자료·회사 홈페이지·SNS 등)에서는 KIMES와 함께 표기해주세요. 표기 규칙은 → /notation.

변경
> 특별관 단독 표기는 특별관 내부 한정. 외부 자료(보도자료·홈페이지·SNS 등)는 KIMES와 병기. 규칙은 → /notation.

## 8. Usage Rights (`/#/permissions`)

### 8-1. 리드문 톤

기존
> 이 페이지가 BI 가이드를 단순한 디자인 가이드와 구별 짓는 핵심입니다. 누가 무엇을 어디까지 할 수 있는지를 명문화한 매트릭스 + 신청 절차 + 라이선싱 문의 안내입니다.

변경
> 사용 권한·신청 절차·라이선스 안내.

### 8-2. Usage Rights Matrix "그 외 사용 케이스" 본문 톤

기존
> 그 외 사용 케이스 안내 — 위 6개에 해당하지 않는 사용 케이스는 사무국과 협의해주세요. 로고 변형·결합·임의 색상 변경·표기 변형(Kimes/킴스 등)·행사 종료 후 상시 게시 등은 권장하지 않습니다.

변경
> 그 외 케이스는 사무국 협의. 로고 변형·결합·색상 변경·표기 변형(Kimes/킴스 등)·행사 종료 후 상시 게시는 비권장.

### 8-3. "파일 선택 참고" 박스 삭제

**삭제 대상**: "파일 선택 참고 (강제 아님)" 박스 전체

**대체**: 한 줄로 축약하여 Usage Rights Matrix 하단에 배치
> 파일 선택은 → /typography 참고.

### 8-4. Timing 본문 톤

기존
> 행사 종료 후에는 회사 자료에서 KIMES 로고를 제거하거나 "지난 KIMES 2027 참가 (Past Exhibitor)" 표기로 변경합니다.

변경
> 행사 종료 후: 로고 제거 또는 "지난 KIMES 2027 참가 (Past Exhibitor)"로 변경.

### 8-5. Application Process 본문 톤

기존 (3단계 카드 부제)
- 사용 목적·매체·기간·시안 첨부
- 영업일 3~5일 안에 사무국 검토
- 메일 회신 + 추가 협의 사항이 있으면 동봉

변경
- 사용 목적·매체·기간·시안 첨부
- **영업일 3~5일 내 사무국 검토**
- 메일 회신 + 추가 협의 사항 동봉

### 8-6. Non-compliance 본문 톤

기존
> 가이드와 다르게 사용된 경우 사무국이 정정을 요청드릴 수 있습니다. 대부분의 경우 협의로 해결되며, 자세한 사항은 brand 문의로 알려 드립니다.
>
> 궁금한 사항은 → kimes@kimes.kr

변경
> 가이드와 다른 사용이 확인되면 사무국이 정정 요청. 문의: kimes@kimes.kr

### 8-7. Licensing 본문 톤

기존
> "KIMES 공식 파트너", "KIMES 추천 업체" 같은 인증·후원 표현은 별도 라이선스 계약이 필요합니다. 가격·계약 조건은 사무국이 개별 협의 합니다.

변경
> "KIMES 공식 파트너", "KIMES 추천 업체" 등 인증·후원 표현은 별도 계약 필요. 조건은 개별 협의.
